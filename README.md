# Cnode

> Vue.js project

### 技术栈
```
  Vue-cli
  Cnode API
  axios
  element-ui
  vuex
```
### 安装方法

```
# 安装依赖
 
npm install
 
# 在本地运行 浏览器访问：localhost:8080
 
npm run dev
 
# 进行打包
 
npm run build
 
```
